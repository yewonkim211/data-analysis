# Fintech User Spending Pattern Analysis

## 1. Project Overview
This project analyzes user spending patterns using transaction data to identify key behavioral insights.

## 2. Dataset
- Source: Kaggle
- Description: User transaction / card spending data

## 3. Key Questions
- How do users spend money by category?
- Are there noticeable patterns by time or user type?
- Which spending categories account for the highest total amount?
- How does spending vary by time (month / day)?

  
## 4. Tools
- SQL (DBeaver)
- Python (Pandas, Matplotlib)
- GitHub

## 5. Structure
/data      : raw data  
/sql       : SQL queries  
/analysis  : Python analysis & notes
